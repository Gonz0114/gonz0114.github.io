My Tag Notes

[P]
the "P", paragraph tab is the core tag used to make long form text
[H]
The "H" tagline, is used for different headings this will be followed by a number to indicate the heading level.
[UL]
The "UL" tag stands for unordered list, this tag in essence is the tag used to create bullet points, similar to other list tags but different in the sense this list is unordered.
[Strong]
the "Strong" tag it uses for emphasis on a word or series of words, this is essentially a Bold text tag.
[Head]
The "Head" tag is used at the beginning of the HTML document, this is the brain of the document and contains information not seen to the end user.
[Body]
The "Body" tag in contrast to the "head" contains all information that is visable to the end user, majoirty of page content should be fitted within the "Body" Tagline.
